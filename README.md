# Prayasha Nanda — Portfolio

This repository contains the source code for the official landing page of Prayasha Nanda, author of the contemporary soft sci-fi novel _Moon of Yesterday_ and a Computer Science & Engineering student.

The site serves as a central hub for both literary works and technical milestones, bridging the gap between creative storytelling and software development.


It includes:
- Book information and purchase links (Amazon, Notion Press, Goodreads)
- Author bio and social links
- Contact email for verification and inquiries

## Tech Stack

- HTML5
- CSS3 (glassmorphism layout)
- Vanilla JavaScript (tab navigation)
- Font Awesome (icons)

## Deployment

This is a static site.
